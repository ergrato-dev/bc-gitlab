# Setup del Entorno — GitLab CE

## Opciones de Instalacion

| Metodo | Guia | Dificultad | Ideal para |
|--------|------|-----------|------------|
| Docker Compose | [con-docker.md](./con-docker.md) | Facil | Desarrollo y practicas |
| Omnibus Package | Guia futura | Intermedio | Produccion |
| Kubernetes (Helm) | Guia futura | Avanzado | Produccion escalable |

## Prerrequisitos Comunes

- Docker y Docker Compose instalados
- Git instalado
- Minimo 8 GB RAM (4 GB para GitLab CE + 2 GB para Runner + 2 GB sistema)
- 20 GB espacio en disco
