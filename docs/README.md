## Requisitos del Sistema

### Minimos para GitLab CE

- **CPU**: 4 cores
- **RAM**: 4 GB (8 GB recomendado)
- **Disco**: 20 GB SSD
- **Sistema operativo**: Ubuntu Server 24.04 LTS (recomendado)

### Para el Entorno de Desarrollo

- Docker 27+
- Docker Compose 2.32+
- Git 2.46+
- 8 GB RAM minimo (GitLab CE + Runner + Registry)
