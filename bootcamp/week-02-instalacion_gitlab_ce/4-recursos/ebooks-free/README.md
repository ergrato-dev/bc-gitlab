# Ebooks Gratuitos — Semana 02

- [Docker Deep Dive — Nigel Poulton (muestra gratuita)](https://nigelpoulton.com/books/)
  Los primeros capitulos cubren conceptos fundamentales de Docker: imagenes, contenedores, volumenes.

- [GitLab Installation Guide (PDF)](https://docs.gitlab.com/ee/install/)
  Documentacion oficial de instalacion de GitLab CE. Cubre todos los metodos (Omnibus, Docker, K8s, source).

- [GitLab Omnibus Configuration Reference](https://docs.gitlab.com/omnibus/settings/)
  Referencia completa de todas las opciones de configuracion en `gitlab.rb`. Buscable, con ejemplos.
