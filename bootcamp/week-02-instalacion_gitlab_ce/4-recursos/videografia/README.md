## Descripción detallada

### Docker Volumes Explained — TechWorld with Nana (⭐ Recomendado)

El video más claro y conciso sobre volúmenes Docker disponible en YouTube. Explica la diferencia entre named volumes, bind mounts y tmpfs con animaciones. Directamente aplicable a entender por qué `bc-gitlab-data` persiste los datos de GitLab.

**Ver antes de:** la práctica 04 (backup y restore).

### Docker Compose Tutorial — TechWorld with Nana

Tutorial completo de Docker Compose que cubre todo lo que necesitas saber: servicios, redes, volúmenes, variables de entorno, healthchecks y profiles. La instructora es excelente para explicar conceptos complejos con claridad.

**Ver antes de:** la práctica 02 (levantar GitLab).

### GitLab Docker Install — GitLab Unfiltered

Video oficial del equipo de GitLab mostrando la instalación paso a paso con Docker Compose. Aunque puede tener diferencias menores con el entorno del bootcamp (usa versiones ligeramente distintas), el flujo conceptual es idéntico.

**Momento ideal:** Como referencia paralela mientras haces la práctica 02.

### GitLab Backup and Restore — GitLab Unfiltered

Demo oficial del proceso completo de backup y restore. Ver este video antes de la práctica 04 te dará contexto visual de lo que harás.
