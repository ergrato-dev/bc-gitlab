# Webgrafia — Semana 05

- [GitLab CI/CD YAML Reference](https://docs.gitlab.com/ee/ci/yaml/)
  Referencia completa de todas las keywords de `.gitlab-ci.yml`. El recurso mas importante para CI/CD.

- [GitLab Runner Install — Docker](https://docs.gitlab.com/runner/install/docker.html)
  Guia oficial de instalacion y registro de GitLab Runner en Docker.

- [GitLab CI Lint Tool](https://docs.gitlab.com/ee/ci/lint.html)
  Como usar el validador de sintaxis integrado en GitLab (Project → CI/CD → Pipelines → CI Lint).

- [Predefined CI/CD Variables](https://docs.gitlab.com/ee/ci/variables/predefined_variables.html)
  Lista completa de variables predefinidas: `$CI_COMMIT_SHA`, `$CI_COMMIT_REF_SLUG`, `$CI_PIPELINE_ID`, etc.

- [Artifacts Documentation](https://docs.gitlab.com/ee/ci/yaml/#artifacts)
  Todas las opciones de artifacts: paths, exclude, expire_in, when, reports (JUnit, SAST, etc.).

- [Cache Documentation](https://docs.gitlab.com/ee/ci/yaml/#cache)
  Configuracion de cache: key, paths, policy (pull/push), fallback_keys.

- [GitLab Runner Docker Executor](https://docs.gitlab.com/runner/executors/docker.html)
  Configuracion del ejecutor Docker: imagenes, volumes, services, DIND.

- [GitLab CI Examples](https://docs.gitlab.com/ee/ci/examples/)
  Ejemplos oficiales de pipelines para diferentes lenguajes y frameworks.
