# 🎬 Videografía — Semana 01

Los videos están organizados de menor a mayor duración. Se recomienda ver al menos los marcados con ⭐.

## Videos Cortos (≤ 15 minutos)

### ⭐ Git Explained in 100 Seconds — Fireship
**URL**: [youtube.com/watch?v=hwP7WQkmECE](https://www.youtube.com/watch?v=hwP7WQkmECE)
**Duración**: 2 minutos
**Nivel**: Principiante
**Descripción**: Explicación ultra-compacta de qué es Git y por qué importa. Fireship es conocido por sus explicaciones densas y visuales. Ideal como primer contacto o repaso de 2 minutos.

---

### Git en 15 Minutos — Fazt Code
**URL**: [youtube.com/watch?v=pBy1zgt0XPc](https://www.youtube.com/watch?v=pBy1zgt0XPc)
**Duración**: 15 minutos
**Idioma**: Español
**Nivel**: Principiante
**Descripción**: Repaso completo de los comandos más usados de Git en español. Fazt Code es uno de los mejores canales DevOps en español. Perfecto para el primer día de la semana.

---

### Git MERGE vs REBASE — Academind
**URL**: [youtube.com/watch?v=0chZFIZLR_0](https://www.youtube.com/watch?v=0chZFIZLR_0)
**Duración**: 9 minutos
**Nivel**: Principiante-Intermedio
**Descripción**: Visualiza la diferencia entre merge y rebase con animaciones claras. Imprescindible antes de leer la Lección 02 sobre ramas y flujos.

## Videos Medianos (15-30 minutos)

### ⭐ Git & GitHub Crash Course — Traversy Media
**URL**: [youtube.com/watch?v=SWYqp7iY_Tc](https://www.youtube.com/watch?v=SWYqp7iY_Tc)
**Duración**: 32 minutos
**Nivel**: Principiante
**Descripción**: Curso intensivo de Git con ejemplos prácticos en terminal. Cubre init, add, commit, push, pull, branches. Brad Traversy es conocido por sus explicaciones claras y prácticas. El mejor video "un solo video para entender Git".

---

### Git Branches Tutorial — Fireship
**URL**: [youtube.com/watch?v=e2IbNHi4uCI](https://www.youtube.com/watch?v=e2IbNHi4uCI)
**Duración**: 7 minutos
**Nivel**: Principiante-Intermedio
**Descripción**: Tutorial visual de ramas en Git. Animaciones que hacen que el concepto de "puntero a un commit" sea intuitivo.

## Recurso Interactivo (No es Video)

### ⭐ Learn Git Branching — Peter Cottle
**URL**: [learngitbranching.js.org/?locale=es_ES](https://learngitbranching.js.org/?locale=es_ES)
**Tipo**: Juego/simulador interactivo
**Nivel**: Principiante a Avanzado
**Descripción**: La mejor forma de aprender ramas, merge, rebase y comandos avanzados de Git de forma visual. Funciona en el navegador, sin instalar nada. Completa al menos los primeros 4 niveles de "Introducción a Git".

> ⭐ **Recomendado especialmente**: Dedica 30-60 minutos a este recurso. Vale más que cualquier video. Las animaciones te harán entender visualmente qué hace cada comando.
