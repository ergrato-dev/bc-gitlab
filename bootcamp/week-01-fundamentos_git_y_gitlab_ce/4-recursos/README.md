## 📖 Ebooks Gratuitos

| Libro | URL | Qué aprender en él |
|-------|-----|---------------------|
| **Pro Git** (Scott Chacon) | [git-scm.com/book/es/v2](https://git-scm.com/book/es/v2) | Fundamentos completos de Git; capítulos 1-3 para esta semana |
| **Git Internals** (Scott Chacon) | [github.com/pluralsight/git-internals-pdf](https://github.com/pluralsight/git-internals-pdf/raw/master/drafts/peeled.pdf) | Cómo funciona Git por dentro (objetos, refs, packfiles) |
| **GitLab Handbook** | [handbook.gitlab.com](https://handbook.gitlab.com/) | Cómo trabaja GitLab internamente; guías de Git y flujos |

**Lectura recomendada para esta semana**: Pro Git, capítulos 1 (Inicio), 2 (Fundamentos) y 3 (Ramas).
