## 💡 Tips para Usar Esta Webgrafía

- Cuando tengas un error, primero busca en **ohshitgit.com** o **gitexplorer.com**
- Para referencia de comandos, usa siempre **git-scm.com/docs** (no terceros)
- Para features de GitLab, usa siempre **docs.gitlab.com** (la documentación oficial es muy buena)
- Dedica al menos 30 minutos a **learngitbranching.js.org** — vale más que cualquier video
