# Ebooks Gratuitos — Semana 01

- [Pro Git (2nd Edition) — Scott Chacon, Ben Straub](https://git-scm.com/book/es/v2)
  Libro oficial de Git. Gratuito, disponible en espanol y multiple formatos (PDF, EPUB, web). Cubre desde fundamentos hasta internals de Git.

- [GitLab Handbook](https://handbook.gitlab.com/)
  Documentacion interna de GitLab sobre como trabajan. Contiene guias de Git, CI/CD, y flujos de trabajo.

- [The DevSecOps Playbook](https://about.gitlab.com/resources/ebook-devsecops/)
  Ebook oficial de GitLab sobre DevSecOps con pipelines integrados.
