# Semana 03 — Proyectos, Grupos y Organización

## Objetivos

- Crear y gestionar proyectos en GitLab
- Organizar proyectos usando grupos y subgrupos
- Configurar visibilidad y permisos
- Implementar estrategias de branching
- Configurar proteccion de ramas

## Requisitos Previos

- GitLab CE instalado y funcional (Semana 02)
- Git basico (Semana 01)
